# Decent_TallyBox_Android
Decent TallyBox 2.932 is an innovative cryptocurrency wallet developed to facilitate digital transactions with simplicity and efficiency. It operates on its own cutting-edge Directed Acyclic Graph (DAG) protocol, reducing transaction costs per transaction as a layer-2 ledger system.

# Requirements
To run "Decent Tallybox" wallet application: 
- ensure you have Android 8.0 or higher installed.
